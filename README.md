# 🍼 Baby Duccio's Baby Shower Invitation

An elegant, high-fashion, and fully responsive digital web invitation designed for **Baby Duccio's Baby Shower Celebration**. This single-page application features interactive micro-interactions, custom geometric vector styles, a real-time countdown timer, and functional social sharing features.

## 🎨 Key Features

*   **Premium Visual Layout:** Integrates customized artwork (`image_48259b.png`) into a seamless, modern layout with soft ambient lighting effects.
*   **Interactive Fluid Sizing:** Dynamically scaled baby blocks spelling out **B-A-B-Y** with a custom blue ribbon badge that repositions beautifully across mobile, tablet, and desktop screens.
*   **Live Countdown Timer:** Keeps track of the exact days, hours, minutes, and seconds remaining until the celebration.
*   **Themed Event Details Section:** Displays event logistics cleanly using a custom, no-SVG dashed "blanket-stitch" card design, complete with color swatch circles for the dress attire.
*   **Smart Social Sharing:** Includes integrated sharing links for Facebook Messenger, WhatsApp, and a dynamic "Copy Link" clipboard action with visual toast confirmation feedback.
